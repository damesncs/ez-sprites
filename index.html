ez-sprites

ez-sprites is a no-frills library for creating 2D games with the JavaScript Canvas API.

  • Breakout demo - left/right arrow keys to move paddle
  • Snake demo - arrow keys to move snake
  • Simple physics demo - click and drag to throw balls
  • Pong demo - up/down arrow keys for right paddle, W and S keys for left paddle
  • Multi-state SVG sprite demo - arrow keys to move character
  • Multi-state SVG sprite with simple physics demo - arrow keys to move character
